# 📈 NIFTY 50 Trading Performance Analytics Dashboard

## 📌 Project Overview

This project performs end-to-end data analytics on historical **NIFTY 50** stock market data to uncover market trends, volatility, trading behavior, and business insights. The analysis includes data cleaning, feature engineering, exploratory data analysis (EDA), key performance indicators (KPIs), and professional visualizations to support investment decision-making.

---

## 🎯 Objectives

- Analyze historical NIFTY 50 market performance
- Perform data cleaning and preprocessing
- Engineer new analytical features
- Conduct exploratory data analysis (EDA)
- Build business KPIs
- Visualize market trends and volatility
- Generate actionable business insights

---

## 📂 Dataset

- **Source:** Yahoo Finance
- **Index:** NIFTY 50 (^NSEI)
- **Time Period:** 2010–2025
- **Records:** ~3,900 daily observations
